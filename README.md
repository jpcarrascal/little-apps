# little-apps

Small, focused utilities built with AI/Copilot. Each app lives in its own folder and can be opened directly from `index.html`.

## Apps

- [DDGTestProjectForm](./DDGTestProjectForm/)  
	A structured writing form for multi-question responses with checklist prompts, per-field word counts/limits, autosave to `localStorage`, copy buttons, and a print/edit toggle.

- [Fibonacci-words](./Fibonacci-words/)  
	A line-by-line text helper that shows word and character counts, highlights the active line, and color-codes lines based on whether the word count is a Fibonacci number.

- [MIDI-CC-Monitor](./MIDI-CC-Monitor/)  
	A browser-based Web MIDI monitor that lists available MIDI input devices and logs incoming MIDI messages (`status`, `data1`, `data2`) in real time.

## Quick start

1. Open [index.html](./index.html) in a browser.
2. Click an app link to launch it.
